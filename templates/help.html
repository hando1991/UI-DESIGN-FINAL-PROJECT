Analyzing a Balance Sheet: Our User Interface Design Final Project
About the Project
This web application serves as our final project for our User Interface Design course with Professor Lydia Chilton. The project consists of building a web application that teaches a user about a particular topic through interactive, iterative learning.
About our Project
Our Subject
We chose to teach our user how to analyze balance sheets since it is a useful topic for young adults to get acquainted with. Additionally, analyzing a balance sheet is a topic that is complicated enough to teach through interactive learning, but also sufficiently easy where it would not take the user more than a few minutes to get through the material.
Our Process
Before settling on analyzing balance sheets, we explored two other topics: Clinical Trials in Drug Design, and Meditation. With the help of our Teaching Assistant Lisa we decided Balance Sheets was the most appropriate topic for the scope of the project, and started collaboratively building this web application through a Github repository.
